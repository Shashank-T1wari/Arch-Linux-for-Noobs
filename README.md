# Introduction

## *Understanding Linux*

Linux is an __open-source__ operating system kernel that forms the foundation of many modern operating systems known as __Linux distributions (or distros)__. Created by __Linus Torvalds__ in __1991__, Linux is widely used in servers, desktops, mobile devices, embedded systems, and cloud infrastructure. Its flexibility, stability, and security make it one of the most powerful and reliable operating systems available today.

Unlike proprietary operating systems, Linux allows users to freely view, modify, and distribute its source code under the __GNU General Public License (GPL)__. This openness has led to a large global community that continuously improves and maintains the system. Linux distributions combine the Linux kernel with system utilities, package managers, and desktop environments to provide complete operating systems tailored for different users and purposes.

Learning Linux helps users understand how operating systems work internally, manage system resources efficiently, and gain greater control over their computing environment.

Learning Linux also grants you __FREEDOM__ from the constraints of proprietary software like Windows and MacOS which often also track your data and limit your control over your own computer. With Linux, you can customize your system to your liking, choose from a wide variety of software, and have the confidence that you are in control of your own digital experience.

## *Arch Linux Distribution*

Why Arch Linux? So you can confidently say __"I use Arch btw"__ and flex on all the noobs who use other distros. Just kidding, but seriously, Arch Linux is a great choice for users who want to learn Linux from the ground up and have complete control over their system. (Become a true Eleutheromaniac)

Arch Linux is a lightweight and highly customizable Linux distribution designed for simplicity, flexibility, and user control. It follows a *rolling release model*, meaning the system receives continuous updates rather than periodic major releases.

One of the defining philosophies of Arch Linux is **KISS (Keep It Simple, Stupid)**. Instead of hiding complexity behind heavy graphical tools, Arch provides a minimal base system that users build upon according to their needs. This approach allows users to create a system that contains only the software they require.

Because of its manual installation process and hands-on configuration, Arch Linux is often recommended for users who want to deeply understand how Linux systems work. Through this process, users gain valuable knowledge about system components, package management, and Linux administration.
